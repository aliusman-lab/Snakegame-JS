# 🐍 Snake Game (JavaScript)

## 📖 Description
A classic Snake Game built using HTML, CSS, and Vanilla JavaScript. The game uses a dynamic grid system where the snake moves continuously, eats food to grow, and ends when it hits the wall. The project focuses on learning DOM manipulation, arrays, game loops, event handling, and JavaScript logic.

## ✨ Features
- Dynamic game board
- Snake movement using arrow keys
- Random food generation
- Snake grows after eating food
- Wall collision detection
- Game Over screen
- Restart game functionality
- Responsive grid layout
